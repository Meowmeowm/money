# iPhone 快捷指令搭建步骤（截屏 → 选金额 → 一键记账）

目标体验：看到支付成功页 → 长按操作按钮 → 弹一个「选金额」列表 → 自动打开记账页且金额已填 → 点个分类完成。全程 ≤4 次触击。

## 搭建步骤（快捷指令 App）

新建快捷指令，命名「记一笔」，依次添加以下操作：

1. **截屏**（Take Screenshot）
2. **从图像中提取文本**（Extract Text from Image），输入选「截屏」
3. **匹配文本**（Match Text）
   - 正则：`\d+(\.\d{1,2})?`
   - 输入：上一步的文本
4. **从列表中选择**（Choose from List）
   - 列表：上一步的「匹配项」
   - 这一步会弹出识别到的所有数字，手动点你要的金额
5. **文本**（Text），内容填：
   ```
   https://你的域名.vercel.app/#amount=已选取的项目
   ```
   （「已选取的项目」用变量插入，来自第 4 步）
6. **打开 URL**（Open URLs），输入选第 5 步的文本

## 绑定到操作按钮（iPhone 15 Pro Max）

设置 → 操作按钮 → 选「快捷指令」→ 选「记一笔」。

之后任何支付完成页：长按操作按钮 → 点金额 → 进 App 点分类，搞定。

## 可选进阶

- **不想弹列表**：如果你的支付页金额总是最大的那个数，可在第 3、4 步之间加「计算统计信息」取最大值，直接跳过选择弹窗（识别错了进 App 里改一下金额即可——退格键改起来也快）。
- **直接入账不进 App 确认**：URL 加 `&save=1`，会直接记到「其他」分类，事后在明细里改分类。适合赶时间的场景。
- **常用分类专属指令**：多建几个快捷指令，如「记打车」用 `...#amount=已选取的项目&cat=transport`，打开后分类已选好，再点一下保存即可。
